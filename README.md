## 🏀 概要

「6th Man」は、バスケットボールの控え選手向けに自主練習を支援するWebアプリです。

私は高校時代、バスケットボール部で控え選手として活動していました。当時は試合に出場する機会が少なく、「自主練習をしたいが、何をどのように練習すればよいのか分からない」という悩みを抱えていました。

そこで、自分なりに練習方法を調べて試行錯誤を重ねながら自主練習を継続した結果、レギュラーメンバーとして試合に出場できるようになりました。

この経験から、過去の自分と同じように自主練習に悩む選手を支援したいと考え、本アプリを開発しました。

ユーザーの課題や利用可能な時間、練習環境に応じて最適な自主練習メニューを提案し、継続的な成長をサポートすることを目的としています。

---

## ✨ 主な機能

### 練習メニュー診断

- 伸ばしたいスキル
- 練習時間
- 利用可能な設備

の3つの質問から最適な自主練習メニューを提案します。

### 練習タイマー

提案された練習メニューごとにタイマーを表示し、練習の進行をサポートします。

### 成長カレンダー

日々の練習記録を保存し、継続状況を可視化できます。

### スキル評価機能

シュート、ドリブル、ディフェンスなどの能力を自己評価し、レーダーチャートで可視化できます。

## 💡 工夫した点

### ユーザー視点での課題設定

自身のバスケットボール経験を基に、「自主練習の方法が分からない」という課題に着目しました。

### 実用性を意識した設計

- 練習メニュー提案
- タイマー機能
- 練習記録
- スキル管理

を一つのアプリに統合しました。

### 継続を支援する仕組み

自主練習は継続が重要であるため、練習記録カレンダーやスキル可視化機能を実装し、モチベーション維持を支援できるよう工夫しました。
